# Superstore Sales Performance Dashboard

This Power BI dashboard provides an analytical overview of Superstore sales across different regions, product categories, and subcategories. It highlights key metrics to evaluate sales performance and profitability.

---

## Key Metrics
- **Total Sales:** 2.30M  
- **Total Profit:** 286.40K  
- **Total Quantity Sold:** 38K  
- **Average Discount:** 0.16  
- **Profit Margin:** 0.12  

---

## Key Insights
- **Technology** products lead in sales, followed by **Furniture** and **Office Supplies**.  
- **West Region** shows the highest total profit.  
- Sales show a steady upward trend between 2014 and 2017.  
- Subcategories like **Phones**, **Storage**, and **Chairs** are top performers.  

---

##  Tools Used
- **Power BI Desktop**
- Data Modeling (Relationships, DAX, Measures)
- KPI Cards, Bar Charts, Tree Maps, and Trend Lines
- Interactive Slicers (Region, Category)
